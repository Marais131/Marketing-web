## 🎯 整合特色

### Gallery4 組件特性
- **現代化輪播**: 使用 Embla Carousel 提供流暢的滑動體驗
- **響應式設計**: 在不同設備上自動調整佈局
- **觸控支援**: 支援滑動和拖拽操作
- **鍵盤導航**: 支援方向鍵控制
- **無障礙設計**: 完整的 ARIA 標記

## 🔧 技術實現

### 組件架構
```typescript
// 主要組件
<Gallery4 
  title="師資陣容"
  description="認識我們優秀的師資團隊..."
  items={gallery4Teachers}
/>

// 輪播控制
<Carousel
  setApi={setCarouselApi}
  opts={{
    breakpoints: {
      "(max-width: 768px)": {
        dragFree: true,
      },
    },
  }}
>
```

### 響應式配置
- **桌面**: 顯示多個卡片，支援左右導航
- **平板**: 適中尺寸，保持可讀性
- **手機**: 單列顯示，支援滑動操作

### 效能優化
- **圖片懶載入**: `loading="lazy"` 和 `decoding="async"`
- **程式碼分割**: 按需載入輪播組件
- **動畫優化**: 使用 CSS transform 而非改變佈局

## 📱 使用方式

### 基本使用
```tsx
import { Gallery4 } from "@/components/ui/gallery4"

const teachers = [
  {
    id: "teacher-1",
    name: "老師姓名",
    title: "職稱",
    role: "角色",
    description: "描述",
    image: "/path/to/image.jpg"
  }
]

<Gallery4 
  title="師資陣容"
  description="描述文字"
  items={teachers}
/>
```

### 進階配置
```tsx
<Gallery4 
  title="自定義標題"
  description="自定義描述"
  items={customItems}
  // 組件內部支援自定義樣式
/>
```
