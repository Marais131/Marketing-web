## 🔧 技術實現

### 組件特性
- **打字速度**: 80ms/字符
- **等待時間**: 2500ms
- **刪除速度**: 50ms/字符
- **游標樣式**: ▌ (方塊游標)
- **循環播放**: 啟用

### 樣式配置
```typescript
className="text-2xl md:text-3xl lg:text-4xl font-bold text-[#3CB1B6]"
cursorClassName="text-[#2A7DB1] ml-2"
```

### 響應式設計
- **桌面**: 大尺寸文字和圖標
- **平板**: 中等尺寸
- **手機**: 小尺寸，單列佈局

## 🚀 使用方式

### 基本使用
```tsx
import { Typewriter } from "@/components/ui/typewriter"

<Typewriter
  text={["文字1", "文字2", "文字3"]}
  speed={80}
  className="text-blue-500"
/>
```

### 進階配置
```tsx
<Typewriter
  text={["自定義文字"]}
  speed={50}
  initialDelay={1000}
  waitTime={2000}
  deleteSpeed={30}
  loop={true}
  showCursor={true}
  cursorChar="|"
  className="text-xl font-bold"
/>
```
